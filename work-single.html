GITHUB: A WALKTHROUGH

Workshop




GitHub is a code sharing and publishing service. Github is one of the most well known and widely used platforms for version control. GitHub is an American company that provides hosting for software development version control using Git which is useful for any software developerâ€¦Github is a subsidiary of Microsoft. Hence we organized workshop Introduction on Github and Git on 27th July 2019 where students were introduced to the basics of the Github, Git, Github Pages and various Git commands.



The Workshop was conducted by Mr. Ashish Salunke and Mr.Varun Pilakavil.



Mr. Ashish Salunke is a Fourth Year Computer Engineering Undergraduate, Intern at CoreView Systems, Pune. He is a Former Intern at MITU Skillologies. He is also a Former Chairperson at PCCOER ACM Student Chapter. He has Special Interests in Natural Language Processing and Deep Learning.


Mr.Varun Pilakavil is also a Fourth Year Computer Engineering Undergraduate. He has programming skills in programming languages like C++, Python, Java, MySQL, and Git. He is a Former Vice-Chairperson at PCCOER ACM Student Chapter.


The Git workshop started off with the basic Git commands with introductory concepts including branching, merging, repository and repository creation, etc. were covered. With the basic command-line out of the way, the internals of Git was covered next. The internals of Git was explained, by building a model of a Git like version control system from scratch. The discussion started with building a single-file version control system, and then adding support for commit logs, branching, merging, and finally multi-file support. With a strong foundation on using the Git repository locally, the mechanism for sharing changes with other developers was covered next. Different types of merge scenarios were also discussed. Repository URLs, cloning repositories and pushing changes to remote repositories was covered next. Fetching changes, also introduced the concept of remote-tracking branches. Effective usage of Git, within a team of developers, was discussed next. Firstly, amending commits and rebasing was discussed. Then Next the centralized workflow and forking workflow was discussed with a demo. Github Pages was also discussed for making a portfolio website.
